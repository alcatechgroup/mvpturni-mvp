# packages/design-tokens

Fonte **única** dos design tokens do Turni (cor, tipografia, espaçamento, etc.), a partir da qual se geram artefatos para os dois frontends heterogêneos: tema **Dart** (WebApp Flutter) e **CSS/Blade** (Backoffice) — ADR-003. Elimina a duplicação de tokens que o PDR-003 teme.

**Stub nesta fase (STORY-006):** os tokens e o pipeline de geração são alimentados pelo Designer a partir do **DDR-001** (Design System), aplicados na **STORY-008**. Por ora, este diretório só reserva o lugar previsto pela ADR-003.
