# contracts/

Contrato de API do Turni (OpenAPI). Fonte única que descreve a API pública (`apps/api`) e a partir da qual se gera o **cliente Dart** consumido pelo WebApp Flutter (ADR-003).

**Vazio nesta fase (STORY-006):** o contrato é preenchido quando os primeiros endpoints existirem (EPIC-001 em diante). Por ora, este diretório só reserva o lugar previsto pela ADR-003.
